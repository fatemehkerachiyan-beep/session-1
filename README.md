آزمایش یک :
عنوان آزمایش: چشمک‌زن LED با استفاده از آردوینو
هدف:
هدف این آزمایش روشن و خاموش کردن یک LED با فواصل زمانی مشخص (۳ ثانیه) با استفاده از پایه‌های خروجی آردوینو است.
توضیح برنامه:
در این برنامه از پایه‌ی شماره 8 آردوینو به عنوان خروجی استفاده شده است. ابتدا در تابع setup() که فقط یک بار اجرا می‌شود، پایه 8 به عنوان خروجی معرفی می‌گردد. سپس در تابع loop() که به صورت مداوم تکرار می‌شود، ابتدا LED روشن شده (HIGH) و به مدت ۳۰۰۰ میلی‌ثانیه (۳ ثانیه) روشن باقی می‌ماند. سپس LED خاموش شده (LOW) و دوباره به مدت ۳ ثانیه خاموش می‌ماند. این چرخه به طور پیوسته تکرار می‌شود و چشمک‌زدن ایجاد می‌کند.
نتیجه:
LED متصل به پایه 8 آردوینو هر ۳ ثانیه روشن و سپس هر ۳ ثانیه خاموش می‌شود و این عمل به طور مداوم تکرار خواهد شد
